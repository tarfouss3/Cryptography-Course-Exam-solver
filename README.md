# Cryptography-Course-Exam-solver
a full solver for every case in the exam of cryptography and blockchain 5 credits version - Howest university . 
This Solver uses the following library made by one of the lecturers at howest : https://pypi.org/project/cryptocourse
can be installed via :
```cmd
python3 -m pip install cryptocourse
```
Solving all the type of exam questions - exercises with multiple options .
